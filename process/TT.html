TRANSFERÊNCIA DE TITULARIDADE

Texto Padrão: Importante verificar pois existe um texto para cada Situação, Se Transferência Concluída ou para Análise, se para fins de Venda ou Locação.

Existem os documentos padrões necessários para cada situação. Em algumas é ainda necessária uma Autorização de Transferência, temos o modelo.

É importante verificar se a pessoa que está solicitando a transferência possui débitos junto à Amazonas Energia, pois caso tenha não será possível realizar o serviço, a não ser que ela o quite.

Importante também verificar se há débitos na UC, verificar se a pessoa irá assumir os débitos. Se não for familiar se for terceiros, digamos alguém que está comprando ou alugando a casa, essa pessoa não terá a obrigação de assumir os débitos da UC.

Para realizar a transferência você deve 1º) entrar na UC, 2º) vai até a casinha azul, 3º) clica em troca de titularidade.

Põe o MOTIVO RESULTADO DO SERVIÇO: 208 No-Normal.

OBS1: É muito importante sempre verificar se a UC é Residencial ou Comercial/Industrial, pois se não for residencial então a transferência será a para ANÁLISE, irá para Manaus. Se cometer o erro de abrir uma transferência Concluída para uma UC COMERCIAL/INDUSTRIAL então este problema será resolvido por Manaus e pode dar suspensão.

OBS2: Se o for o locatário que desejar fazer a transferência de titularidade, então será necessário que haja uma cláusula no contrato de locação na qual o locador autorize que o locatário transfira a titularidade para o nome do locador. Caso não haja essa cláusula, então será necessário que o próprio locador venha até a loja realizar a transferência.
